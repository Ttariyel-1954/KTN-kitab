KTN Kitab

Klassik Test Nəzəriyyəsinə aid kitab

Giriş

Klassik Test Nəzəriyyəsi (KTN) psixometrikanın əsas nəzəriyyələrindən biridir və testlərin keyfiyyətini qiymətləndirmək üçün vacib alətlər təqdim edir.

Bu kitab klassik test nəzəriyyəsinin əsas prinsiplərini, metodlarını və praktik tətbiqlərini əhatə edir. Kitab həm nəzəri biliklər, həm də praktik nümunələrlə zənginləşdirilmişdir.

Qeyd: Bu kitab psixologiya, təhsil və statistika sahələrində çalışan mütəxəssislər üçün nəzərdə tutulmuşdur.

1. Klassik Test Nəzəriyyəsinin Əsasları

1.1 Tarixi Inkişaf

Klassik test nəzəriyyəsi XX əsrin əvvəllərində formalaşmağa başlamışdır. Bu nəzəriyyənin əsasını qoyan əsas şəxslər arasında Charles Spearman, Louis Thurstone və digər tanınmış psixometriklər var.

1.2 Əsas Fərziyyələr

KTN-nin üç əsas fərziyyəsi var:

  • Müşahidə olunan bal = Həqiqi bal + Xəta
  • Xətaların orta değeri sıfırdır
  • Həqiqi bal və xəta arasında korrelyasiya yoxdur
X = T + E
burada: X - müşahidə olunan bal, T - həqiqi bal, E - xəta

2. Etibarlılıq Nəzəriyyəsi

2.1 Etibarlılığın Tərifi

Etibarlılıq testin tutarlılığını və dəqiqliyini ölçən əsas göstəricidir. Bu, testin nə dərəcədə stabil nəticələr verdiyini göstərir.

2.2 Etibarlılıq Növləri

Etibarlılığın müxtəlif növləri mövcuddur:

  • Test-retest etibarlılığı
  • Daxili tutarlılıq etibarlılığı
  • Paralel formalar etibarlılığı
  • Yarıya bölmə etibarlılığı
Nümunə: Cronbach's Alpha əmsalı daxili tutarlılığı ölçmək üçün ən çox istifadə olunan metodlardan biridir.

3. Düzgünlük Nəzəriyyəsi

3.1 Düzgünlüyün Mahiyyəti

Düzgünlük testin nə dərəcədə ölçmək istədiyi konstruktu ölçdüyünü göstərir. Bu, testin ən vacib keyfiyyət göstəricisidir.

3.2 Düzgünlük Növləri

Əsas düzgünlük növləri:

  • Məzmun düzgünlüyü
  • Konstrukt düzgünlüyü
  • Kriteriya düzgünlüyü

4. Mədd Analizi

4.1 Mədd Çətinlik İndeksi

Hər məddın çətinlik səviyyəsini müəyyən etmək üçün istifadə olunan statistik göstərici.

p = (doğru cavab verən şəxslərin sayı) / (ümumi şəxslərin sayı)

4.2 Mədd Seçicilik İndeksi

Məddın yüksək və aşağı qrup arasında fərq yarada bilmə qabiliyyətini ölçür.

5. Test Qurulması

5.1 Test Planlaması

Keyfiyyətli test yaratmaq üçün əvvəlcədən planlaşdırma və spesifikasiya hazırlanması vacibdir.

5.2 Mədd Yazılması

Effektiv test məddləri yazarkən nəzərə alınmalı olan əsas prinsiplər:

  • Aydın və dəqiq ifadə
  • Təkərarnormalı vermə
  • Mədəniyyət və dilə həssaslıq
  • Obyektivlik

Nəticə

Klassik Test Nəzəriyyəsi müasir psixometrikanın əsasını təşkil edir və testlərin keyfiyyətini qiymətləndirmək üçün möhkəm nəzəri çərçivə təqdim edir.

Bu kitabda müzakirə olunan prinsip və metodlar praktik test qurulması və təhlilində geniş tətbiq tapır. Gələcək tədqiqatlarda bu sahənin daha da inkişaf etdirilməsi gözlənilir.

Tövsiyə: Bu sahədə daha dərin bilgi əldə etmək üçün müasir psixometrika ədəbiyyatını izləməyiniz tövsiyə olunur.
